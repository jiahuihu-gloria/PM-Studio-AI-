# Demo Assets

This folder contains demo materials for PM Studio AI.

## Demo Video

`Demo.mp4` should be created from a real product walkthrough recording. It is intentionally not committed as a fake or empty file.

Recommended video structure:

1. Show the dashboard workspace, not a chat interface.
2. Enter: `我想做一个 AI 简历筛选产品。`
3. Run the workflow.
4. Walk through the vertical agent pipeline.
5. Open PRD, AI Architecture, KPIs, Risks, and Evaluation.
6. Close with the portfolio signal: strategy, AI architecture, experiments, and risk governance.
