# User Flow

```text
Start
↓
Enter one-sentence product idea
↓
Run PM Agent
↓
Run Research Agent
↓
Run UX Agent
↓
Run AI Architect Agent
↓
Run Data Analyst Agent
↓
Run QA Agent
↓
Review dashboard
↓
Inspect artifacts
↓
Export product brief
```

## Key Interaction Principles

- The user does not chat with the AI.
- The user initiates a structured workflow.
- The system shows progress through agents.
- Outputs are organized as artifacts, not messages.
- Each artifact has a product decision purpose.
